PROJECT TITLE:Bomberman
PURPOSE OF PROJECT:AP Comp Sci 2016-17 S2
VERSION or DATE: 1.0
HOW TO START THIS PROJECT:RClick GameMain > Void Main > OK
AUTHORS:Daniel Delaney
USER INSTRUCTIONS: P1 - arrow Keys and Ctrl, P2 - WASD and SpaceBar

Player 1:
UP ARROW - move up
DOWN ARROW - move down
RIGHT ARROW - move right
LEFT ARROW - move left
CTRL - place bomb


Player 2:
W - move up
S - move down
A - move left
D - move right
SPACEBAR - place bomb
